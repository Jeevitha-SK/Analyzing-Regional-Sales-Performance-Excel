# Regional-Sales-Analysis-Excel

## Project Overview

This project focuses on analyzing regional sales performance for ABC Electronics using Microsoft Excel. The goal is to clean the data, perform analysis, and build an interactive dashboard to generate business insights.

## Skills Gained

* Data cleaning and preprocessing
* Excel formulas and calculations
* Searching and filtering data
* Pivot tables and slicers
* Charts and visualization
* Regression analysis
* Dashboard creation

## Tools Used

Microsoft Excel

## Key Tasks Performed

* Cleaned and standardized data using text functions
* Calculated sales, profit, and discount metrics
* Created pivot tables for sales analysis
* Built charts to visualize trends
* Performed regression analysis between discount and sales
* Designed an interactive dashboard

## Project Outcome

* Identified top-performing regions and product categories
* Analyzed the impact of discounts on sales
* Built an interactive dashboard for business insights
